PREFERRED INSTALLATION

Review and merge the selected-bowler dashboard pull request. GitHub Pages and the results updater
will use the new files after the merge.

FULL REPLACEMENT STEPS (GITHUB DESKTOP BACKUP METHOD)

1. Download and unzip the replacement package.
2. Open jack-wix-junior-gold in GitHub Desktop.
3. Select Repository > Show in Finder.
4. Press Command + Shift + . so the hidden .github and .git folders are visible.
5. DO NOT delete or replace the existing .git folder.
6. Copy every file and folder from the unzipped replacement into the local
   repository folder. Choose Replace or Merge when macOS asks.
7. Confirm the replacement includes data/bowlers.json,
   scripts/build_2025_archive.py, scripts/update_results.py, app.js,
   index.html, styles.css, VERSION, .github/workflows/update-results.yml, and
   .github/workflows/bump-version.yml.
8. Commit with this summary:
   Personalize dashboard for selected bowlers
9. Click Push origin.
10. On GitHub, open Actions > Update Junior Gold dashboard > Run workflow.
11. Wait for a green check, then open the Pages site and hard-refresh with
    Command + Shift + R.
