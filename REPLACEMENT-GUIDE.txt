PREFERRED INSTALLATION

Review and merge the selected-bowler dashboard pull request. GitHub Pages and the results updater
will use the new files after the merge.

FULL REPLACEMENT STEPS (GITHUB DESKTOP BACKUP METHOD)

1. Download and unzip the replacement package.
2. Open jack-wix-junior-gold in GitHub Desktop.
3. Select Repository > Show in Finder.
4. Press Command + Shift + . so the hidden .github and .git folders are visible.
5. DO NOT delete or replace the existing .git folder.
6. Copy every file and folder from the unzipped replacement into the local
   repository folder. Choose Replace or Merge when macOS asks.
7. Confirm the replacement includes data/bowlers.json,
   scripts/build_2025_archive.py, scripts/update_results.py, app.js,
   index.html, styles.css, and .github/workflows/update-results.yml.
8. Commit with this summary:
   Personalize dashboard for selected bowlers
9. Click Push origin.
10. On GitHub, open Actions > Update Junior Gold dashboard > Run workflow.
11. Wait for a green check, then open the Pages site and hard-refresh with
    Command + Shift + R.

WHAT THIS UPDATE ADDS

- Search any published U18 Boys bowler by name or hometown.
- Select a state to immediately list every published bowler from that state.
- Switch between all 1,341 final 2025 profiles and the live 2026 field.
- Open shareable profiles using the year and USBC ID in the URL.
- Update the page title, hero, statistics, scores, charts, cut copy, comparison,
  tournament path, guide, and share text for the selected bowler.
- Show qualifying blocks, source reports, and 2025-versus-2026 comparisons only
  when those data are available.
- Collapse every dashboard section on page load.
- Hide or show individual sections from the display manager near the top; those
  visibility choices persist in the visitor's browser.
- Disable Jack-only or otherwise unavailable sections when another profile is
  active, while keeping their headings visible in the display manager.
- Never infer missing individual game scores. Derived 2025 Day 3 totals are
  explicitly labeled and use adjacent official cumulative totals only.
