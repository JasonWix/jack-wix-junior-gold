PREFERRED INSTALLATION

Review and merge the selected-bowler dashboard pull request. GitHub Pages and the results updater
will use the new files after the merge.

FULL REPLACEMENT STEPS (GITHUB DESKTOP BACKUP METHOD)

1. Download and unzip the replacement package.
2. Open jack-wix-junior-gold in GitHub Desktop.
3. Select Repository > Show in Finder.
4. Press Command + Shift + . so the hidden .github and .git folders are visible.
5. DO NOT delete or replace the existing .git folder.
6. Copy every file and folder from the unzipped replacement into the local
   repository folder. Choose Replace or Merge when macOS asks.
7. Confirm the replacement includes data/bowlers.json, data/registration.json,
   scripts/build_2025_archive.py, scripts/build_2026_registration.py,
   scripts/update_results.py, app.js,
   index.html, styles.css, VERSION, .github/workflows/update-results.yml, and
   .github/workflows/bump-version.yml.
8. Commit with this summary:
   Add all-division Junior Gold bowler search
9. Click Push origin.
10. On GitHub, open Actions > Update Junior Gold dashboard > Run workflow.
11. Wait for a green check, then open the Pages site and hard-refresh with
    Command + Shift + R.

WHAT THIS UPDATE ADDS

- Search any registered 2026 U12, U14, U16, or U18 boys or girls bowler by
  name or hometown.
- Merge qualifying scores and standings into registered profiles as Bowl.com
  publishes each division report.
- Select a state to immediately list every published bowler from that state.
- Switch between all eight 2026 divisions and the 1,341-profile final 2025
  U18 Boys archive.
- Open shareable profiles using the year, division, state, and bowler ID in the URL.
- Update the page title, hero, statistics, scores, charts, cut copy, comparison,
  tournament path, guide, and share text for the selected bowler.
- Show qualifying blocks, source reports, and 2025-versus-2026 comparisons only
  when those data are available.
- Collapse every dashboard section on page load.
- Start with the Dashboard guide first and Jack's ball card hidden.
- Hide, show, or drag-reorder individual sections from the display manager near
  the top; order and visibility choices persist in the visitor's browser.
- Provide up/down ordering buttons for phones, touch screens, and keyboards.
- Build the state participant list from the Bowler Explorer's selected year,
  division, and state, including pending rows without posted scores.
- Disable Jack-only or otherwise unavailable sections when another profile is
  active, while keeping their headings visible in the display manager.
- Never infer missing individual game scores. Derived 2025 Day 3 totals are
  explicitly labeled and use adjacent official cumulative totals only.
- Display the dashboard version at the bottom of every page and automatically
  increase it after each pull request is merged. The included VERSION file is
  v5; merging pull request #6 publishes v6.
- Hide sections that have no source-backed data for the selected bowler,
  including the next-block card, last-qualifier card, personal schedule, ball
  card, comparison, cut, progress, scores, and state leaderboard. Unavailable
  sections are also omitted from the display manager.
- Pin Official results status directly below the Live Tournament Dashboard
  header. It remains collapsible but cannot be hidden or displaced by a saved
  drag-and-drop order when 2026 source status is available.
